About Our Server

Survival

This server is a vanilla survival style experience, modeled off great servers like HermitCraft and MindCrack. Building, resource gathering and cooperative play between players is the goal. Spawn, nether hub and community projects.

Learn more

Paper

This server runs Paper, a high performance version of the Minecraft server software. This allows us to use plugins for a more streamlined experience. No game altering plugins are present to preserve the vanilla experience.

Learn more

Server Map

There is a live map of the server, including all dimensions and players! It shows a top down and isometric perspective and can be very handy. It only shows explored chunks to keep with the vanilla theme.
